# Path-of-the-crystal
Las profundidades esconden los Cristales Ancestrales. Como soldado de una unidad de élite, tu misión es recuperarlos mientras enfrentas peligrosas criaturas y superas desafiantes cavernas. Recolecta todos los cristales para avanzar, derrota a tus enemigos y demuestra que tienes lo necesario para completar la misión.
